# amity-square

Pokémon-style isometric park explorer on Zo Space.

**Live:** https://etok.zo.space/amity-square

## What it is

A playable 34×26 tile park with:
- WASD / arrow key movement
- Tile-based collision (water, trees, fences, bushes)
- Smooth camera following with lerp
- Pixel-art Pichu follower with path lag
- Animated water tiles and decorative flowers

## Tech

- Pure canvas rendering, no dependencies
- Self-contained single React component
- Works in any modern browser
